# 水晶珠宝独立站新手起步版

这个项目是一个适合新手的静态独立站模板，专门给水晶珠宝品牌做起步使用。

## 你现在已经拥有的内容
- 首页：品牌主标题、介绍、按钮
- 品牌故事板块：讲清楚你做什么
- 系列展示：放三个产品卡片
- 建站流程：让客户知道你是怎么做的
- 联系方式表单：方便客户咨询

## 你可以这样一步一步改
1. 先换品牌名称
   - 把页面里的“璀璨水晶”换成你的品牌名
2. 换文案
   - 把首页标题、介绍、产品描述改成你的真实内容
3. 换图片
   - 你可以把图片地址改成你的产品照片
4. 加上你的联系方式
   - 电话、微信、邮箱、WhatsApp、Instagram
5. 继续扩展页面
   - 例如：关于我们、产品详情页、FAQ、购物须知

## 你需要的文件
- index.html：页面内容
- styles.css：页面样式
- script.js：小动画和表单交互

## 预览方式
- 直接双击 index.html 打开浏览器查看
- 如果你想要更像正式网站的效果，可以继续部署到网站空间
