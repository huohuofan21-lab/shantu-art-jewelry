素材上传说明

请把品牌 LOGO 与产品图片放到本目录（assets/images/）。上传前请按以下规则命名和准备：

命名规范示例：
- logo.svg
- logo-light.png (浅色背景版本，可选)
- product-<slug>-main.jpg (主图，横向或方图)
- product-<slug>-detail-1.jpg
- painting-<slug>-01.jpg
- portrait-zhang-junjie.jpg

要求与建议：
- LOGO 优先 SVG；若为 PNG，请提供透明底版本
- 单张图片建议不超过 8MB
- 我会自动生成 WebP 压缩版与 3 种缩略图（大、中、小）并保存为：
  - <name>-lg.webp  (1200px)
  - <name>-md.webp  (800px)
  - <name>-sm.webp  (400px)

上传方式：
1. 直接把文件复制到此目录（推荐）
2. 或在本对话上传附件，我会代为保存到此目录

请在上传后告诉我：
- 总共上传了多少张图片
- 每张图对应的页面/产品（可按文件名说明）
- 是否需要我进行裁切或色彩微调（例如：统一为暖色基调）
